# Raízes Da Esperança

Projeto desenvolvido como TCC na ETEC durante o 3º ano do ensino médio.

## Sobre

Raízes Da Esperança é um jogo desenvolvido como projeto de conclusão de curso, focado em atmosfera, narrativa e exploração.

## Status

O código-fonte original foi perdido, porém a build jogável foi preservada.
